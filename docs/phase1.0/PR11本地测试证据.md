- **strategy_id 缺失 → 422 + reason_code=MISSING_STRATEGY_ID**：已覆盖（test_missing_strategy_id_returns_4xx_not_500）。

- **strategy disabled → 422 + STRATEGY_DISABLED**：已覆盖（test_webhook_strategy_disabled_returns_422，补测完成）。
- **不同策略：仓位/冷却/同向去重互不影响**：已覆盖（test_risk_position_isolated_by_strategy_id 覆盖仓位隔离；冷却与同向去重按 strategy_id 在 risk_manager 中隔离，由既有 test_risk_* 覆盖）。

- **execution_events 记录 FINAL_FAILED + reason_code**：已覆盖（同上）。
- **多策略 + ALLOW_POSITION_SCHEMA_DOWNGRADE=true → 启动期 fail-fast（MULTI_STRATEGY_POSITION_DOWNGRADE_FORBIDDEN）**：已覆盖（test_multi_strategy_allow_position_schema_downgrade_fails_startup）。
- **007 downgrade 没有 ALLOW_DATA_LOSS=true → RuntimeError**：已通过脚本 scripts/verify_migration_downgrade_gate.sh 可审计验证（见第 5 节）。
